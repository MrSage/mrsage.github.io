NAuth

A Bluetooth keyboard MFA application for iOS


Security protocols at work were causing me to pull my phone out too often. I wanted an easier way to handle MFA. My idea was a phone app that could enter the code for me using Bluetooth. Much to my dismay I learned that iOS prevents this behavior. However, I still learned a lot and built a useful application.





django_contracts

A lightweight library for enforcing API contracts in Django


I often find myself rewriting the same boilerplate code over and over again. I write and maintain this library and use it in all of my Django projects. It uses the power of Django forms to provide a lot of the convenience of Django Rest Framework without binding you to another framework.





Shelf

A free tier Jira replica


The year is 2018, project management software is everywhere but there is not a decent free tier option. Jira is too large to host on a Raspberry Pi, but I need a solution for a project I wanted my friends to collaborate on. I wrote a fully working free tier version of Jira using Python3 and Django. Shelf was my first fully featured application I released on the web. Unfortunately it wasn't enough to inspire collaboration within my friends.
